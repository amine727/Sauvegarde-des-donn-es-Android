#  LAB 14 – Sauvegarde des données Android

##  Description
Ce projet Android (Java) montre différentes méthodes de stockage local :
- SharedPreferences (données simples)
- EncryptedSharedPreferences (données sécurisées)
- Stockage interne (fichiers texte + JSON)
- Cache
- Stockage externe (app-specific)
##  Fonctionnalités

- Sauvegarde / chargement des préférences (nom, langue, thème)
- Stockage sécurisé d’un token (chiffré)
- Sauvegarde et lecture d’un fichier JSON (liste d’étudiants)
- Écriture/lecture fichier interne
- Utilisation du cache
- Stockage externe

---

##  Sécurité
- Utilisation de `EncryptedSharedPreferences`
- Pas de stockage en clair des données sensibles
- Pas de logs de token
